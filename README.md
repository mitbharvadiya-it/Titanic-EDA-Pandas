# Titanic Dataset EDA using Pandas

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the Titanic Dataset using Python and Pandas.

## Skills Demonstrated

- Data Cleaning
- Missing Value Handling
- Filtering
- Sorting
- GroupBy Operations
- Feature Engineering
- Pivot Tables
- Exporting Cleaned Data

## Features Created

- Age_Group
- Family_Size
- Status (Alone / With Family)

## Technologies Used

- Python
- Pandas
- NumPy

## Dataset

Titanic Dataset
